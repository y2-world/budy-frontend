# budy-frontend

Vite + React + TypeScript によるフロントエンドアプリケーションです。ダイエットアプリ「Budy」のUI部分を構成しています。

## 公開URL

- 本番環境（Vercel）: https://budy-diet.vercel.app
- 開発環境: http://localhost:3000

---

## 環境構築

### 必要要件

- Node.js（推奨: v16以上）
- npm or pnpm

### セットアップ手順

```bash
git clone https://github.com/your-username/budy-frontend.git
cd budy-frontend
npm install
npm run dev
